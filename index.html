Data Flow
How Results Are Stored

The Cowboy Polo Circuit uses a hybrid storage model. Live match data is kept off-chain for performance and rapid updating. End-of-season summaries are published to IPFS, and the IPFS hash is placed on-chain on the Base Network for transparency and permanence.

Match → Supabase (off-chain detailed log)

Season Summary → JSON → IPFS

IPFS Hash → Base Smart Contract (immutable)

Leaderboards → Off-chain, public-facing

Tribute → Patronium snapshot per season
